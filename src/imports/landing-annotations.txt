OPPRETT FORESPØRSEL — annotations Artboard
Mobile form artboard: 375 x 900 px (scrollable).
Header

Back arrow left, title center "Opprett forespørsel", help icon right (24x24).
Steg 1 — Grunninfo

Felt: Kategori dropdown (Trevare/Rør/Elektro) — height 48 px.
Tittel/overskrift felt: "Kort tittel av jobben" — input height 48 px placeholder "Eks: Bytte dørhengsler".
Beskrivelse (textarea): height 160 px, placeholder "Beskriv jobben detaljert. Legg ved mål, materialer og ønsket tidsrom."
Tip: small hint tekst under textarea: "Tips: ta bilder og mål for raskere tilbud" — Inter 13 px #6B7280.
Steg 2 — Bilder

Bildeopplaster komponent:
Drag & drop area height 140 px, dashed border #E5E7EB, center upload icon 48x48 and tekst "Dra bilder hit eller trykk for å velge".
Thumbnail preview: 72x72, gap 8 px, max 8 bilder. Each thumbnail: overlay delete icon top-right (16x16).
Komprimering & resizing: anbefalt 2MB max per bilde; show error hvis større.
Interaksjon: tap thumbnail -> opens lightbox (zoom) + add caption (felt under each).
Steg 3 — Lokasjon & tid

Lokasjon input: autocompleted Google Places field with map pin icon left; height 48 px.
Adresse fallback: manual input felt.
Ønsket tidsrom: datepicker (start + end) or ASAP toggle.
Postnummer inferencing: if postnr given, automatically set region filter for matchmaking.
Steg 4 — Preferanser & budsjett (valgfritt)

Budsjett-intervall (min/max) — slider component full width; display numeric fields under (input height 48 px).
Preferanser: checkbox list: "Kun verifiserte fagfolk", "Få faste pris-pakker", "Vis kun leverandører med rating >= 4.5".
Submit area

CTA primary: "Send forespørsel" — full width, height 56 px bg #E07B3E, text white Inter 16 semibold.
Secondary: "Lagre som utkast" — ghost button under.
After submit — confirmation modal

Modal content:
Tittel: "Forespørsel sendt!"
Undertekst: "Vi sender forespørselen til relevante fagfolk i ditt område. Du får svar innen X timer." — Inter 14 px.
CTA: "Gå til mine forespørsler" (solid #17384E) and "Opprett ny forespørsel" (ghost).
Notifications:
Push/web-push allowed prompt (standard browser prompt) — explain benefit.
Form validations & error states

Required fields: Category, Title, Description, Location.
Error styling: red border #EF4444, error text Inter 12 px color #EF4444 under field.
Network/loading state: spinner inside primary CTA, disabled while sending.
Accessibility notes

All interactive elements: min touch target 44x44 px.
Color contrast: ensure text-on-CTA passes WCAG AA (check #E07B3E on white is OK for buttons with white text).
ARIA: use aria-live for submission confirmation, aria-invalid for inputs.
Export settings (form & request screens)

Export mobile artboards at @2x (scale 2) as PNG: filenames request-mobile@2x.png, registration-mobile@2x.png.
Export icons as SVG; images used in hero/illustration as optimized PNG or SVG (vector preferred).
For thumbnails, export web-optimized PNG at 72 DPI, max width 144 px (for retina use 288 px).

CTA "Opprett forespørsel" -> open Request artboard modal (overlay).
"Se hvordan det fungerer" -> open How-it-works modal (3-step).
